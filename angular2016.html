Angular, what is it?

What Javascript MVWhatever framework
Why Support the (frontend) developer to build dynamic frontend apps
How Reusable components: Services, Controllers, Directives

For whom?

If you want to:

  • Separate UI from Internals
  • Easily consume RESTful and RESTlike services
  • Combine multiple datasources in your landscape

The alternatives

  • Ember
  • Knockout
  • React

Why still Angular?

Community, community, community

  • Battle tested: Doubleclick, Google analytics, ReadWriteWeb, Docker, Godaddy, Ford,...
  • Documentation, blogs, stackoverflow,...
  • Plugins and tools: Protractor, karma, batarang, gulp/grunt plugins

What else is there?

  • UI Router: View States, app as statemachine
  • Angular UI: UI components with twitter bootstrap
  • NG Material: UI components with Material design
  • ngtranslate, nganimate, restangular,...
  • Ionic: Mobile app framework and UI components, with Cordova
  • MEAN: MongoDB, ExpressJS, Angular, Nodejs

How do I start?

NG Module


// declare a module
var myAppModule = angular.module('myApp', []);

Reusable Javascript components

Values, services, factories, providers


angular.module('myModule', [])
	.value("myImportantValue","42")
	.service("myMoment", function(){ return new Date(); } );
	.factory("myCounter", function(){var count=0; return function(){alert(count++);}})

Controller

Initialize and track state


angular.module('myModule', [])
	.controller("myController", [ "myCounter" , function(myCounter){
		this.text = "my text";
		this.initializationTime = myMoment();
		this.counter = myCounter;
	})]

Directive

UI Components: unite some logic with some markup design
